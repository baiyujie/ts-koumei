## 快速开始

``` bash
npm install koumei --save
```

``` js
import * as avalon from 'avalon2';
import 'koumei';
// 打印一下会发现组件库已经挂在 avalon 上了
console.log(avalon.components);
```

组件的具体使用方法参考这个项目 [ts-designer](https://github.com/baiyujie/ts-designer/)，独立完善的组件文档会在近期推出

## 浏览器支持
